# 소트인사이드

<br/>

### 문제

**배열을 정렬하는 것은 쉽다. 수가 주어지면, 그 수의 각 자리수를 내림차순으로 정렬해보자.**

<br/>

### 입력

**첫째 줄에 정렬하고자하는 수 N이 주어진다. N은 1,000,000,000보다 작거나 같은 자연수이다.**

<br/>

### 출력

**첫째 줄에 자리수를 내림차순으로 정렬한 수를 출력한다.**

<br/>

<br/>

일단 이 문제는 정렬의 크기나, 출력문의 길이가 굉장히 짧으므로, 시작복잡도를 굳이 생각 안 하고 아무 정렬이나 써서 풀어도 되겠다고 생각했다. 그래서 오늘 배운 퀵소트를 사용해 풀었다.

이 문제는 입력길이도 짧기때문에 입력받은 수를 하나하나 잘라서 String 배열에 넣은 후 int 배열에 for문을 돌려가며 넣으려 했지만 너무 귀찮았고, 무조건 그런걸 해주는 메소드가 있을거라 생각해 찾아보았다.
